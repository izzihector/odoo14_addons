Odoo 12 POS Loyalty

Allow customers to earn and redeem loyalty Points.


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Separate Configuration for loyalty.
  • Loyalty Point Calculation is based on Products / Order Total.
  • Earn Points based on product or POS product category.
  • Send Mails to customers for their loyalty points.

How to use

  • Click on Loyalty for configuration.
  • Points calculation based on 2 ways by Order total or by product.


  • Enable configuration to use Loyalty

  • Loyalty Points based on product or Pos product category


  • Loyalty Points in POS


  • Earned Loyalty

  • Redeem Loyalty in POS

  • Loyalty in Receipt

  • Redeem Loyalty in Backend
